# 🔧 Services — Konfigurasi Layanan Sendbook

Setiap folder di sini berisi dokumentasi & isian konfigurasi untuk satu service.

## Daftar Service

| Folder | Service | Status |
|---|---|---|
| `github/` | GitHub (repo, remote, secrets) | ⬜ |
| `cloudflare/` | Cloudflare (Workers, R2, Pages, DNS) | ⬜ |
| `supabase/` | Supabase (PostgreSQL database) | ⬜ |
| `upstash/` | Upstash Redis (caching) | ⬜ |
| `expo/` | Expo / EAS (mobile app build) | ⬜ |
| `domain/` | Domain sendbook.id | ⬜ |
| `payment/` | Midtrans / Xendit (payment gateway) | ⬜ |
| `template/` | Template kosong untuk service baru | — |

## Cara Pakai

1. Baca folder service yang ingin dikonfigurasi
2. Isi tabel di `README.md` masing-masing
3. Checklist langkah-langkah yang sudah dilakukan
4. Commit perubahan ke git
