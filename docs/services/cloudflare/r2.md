## Usage

Upload via API:
```
POST /api/upload
Content-Type: multipart/form-data
Authorization: Bearer <session_token>
file: <image>
```

Response:
```json
{
  "url": "https://pub-8b6a4088db2c4966974f91de589f6cb9.r2.dev/filename.jpg"
}
```
