# Cloudflare R2 — Image Storage

> Menyimpan foto produk UMKM. S3-compatible, free 10GB.

| Item | Nilai | Status |
|---|---|---|
| **Bucket Name** | `sendbook-products` | ⬜ |
| **Public URL** | `https://pub-xxxxx.r2.dev` | ⬜ |
| **Custom Domain** | (opsional) `images.sendbook.id` | ⬜ |

## Langkah Setup

- [ ] Buka Cloudflare Dashboard → R2
- [ ] Create bucket → nama: `sendbook-products`
- [ ] Settings → Public Access → **Allow Access**
- [ ] (Opsional) Set custom domain untuk R2
- [ ] Binding sudah di `wrangler.toml`:
```toml
[[r2_buckets]]
binding = "SENDBOOK_ASSETS"
bucket_name = "sendbook-products"
```

## Catatan

- Upload gambar via API endpoint `POST /api/upload` (butuh auth)
- File divalidasi: max 5MB, hanya image
- URL gambar bisa diakses publik via R2 public URL
